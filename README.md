# 隐私政策 / Privacy Policy

**应用名称：工具口袋（HTML Pocket）**
**最后更新日期：2026 年 6 月 29 日**

## 中文版

### 概述

「工具口袋」是一款让你在本地运行、保存和管理 HTML 代码的工具类应用。我们高度重视你的隐私。本应用的设计原则是**所有数据都保存在你的设备本地**，我们不收集、不上传、不出售你的任何个人信息。

### 我们收集哪些信息

**我们不收集任何个人信息。**

具体而言：

- 你粘贴、运行、保存的 HTML 代码，仅存储在你的设备本地。
- 你为保存的应用设置的名称和图标，仅存储在你的设备本地。
- 我们没有服务器，不会将上述任何内容上传到云端或任何第三方。
- 我们不收集你的姓名、邮箱、位置、设备标识符或任何可识别身份的信息。
- 我们不使用任何第三方分析、广告或追踪 SDK。

### 数据如何存储

- 你的所有数据（HTML 内容、应用名称、自定义图标）都保存在设备的应用沙盒中。
- 这些数据不会离开你的设备，除非你**主动**使用「分享」功能将某个保存的应用导出为文件发送给他人。
- 当你卸载本应用时，所有本地数据将被一并删除。

### 关于运行 HTML 内容

- 本应用通过系统的网页组件（WebView）运行你提供的 HTML 代码。
- 应用本身不会主动联网。但如果你运行的 HTML 代码中包含指向网络的链接或资源（例如外部图片、脚本），运行该代码时可能会产生相应的网络请求——这取决于你所运行的 HTML 内容本身，与本应用无关。
- 请仅运行你信任来源的 HTML 代码。

### 分享与导入功能

- 「分享」功能由你主动发起，将你选择的应用打包为 `.pocket` 文件，通过你选择的方式（如隔空投送、即时通讯软件等）发送。我们不参与、不经手、不存储这一传输过程中的任何数据。
- 「导入」功能在你打开他人发来的 `.pocket` 文件时，会在导入前向你确认，仅在你同意后才将其内容保存到你的设备本地。

### 儿童隐私

本应用不面向 13 岁以下儿童，也不会主动收集任何儿童的个人信息。

## English Version

### Overview

"HTML Pocket" (工具口袋) is a utility app that lets you run, save, and manage HTML code locally on your device. We take your privacy seriously. The app is designed so that **all data stays on your device** — we do not collect, upload, or sell any of your personal information.

### Information We Collect

**We do not collect any personal information.**

Specifically:

- The HTML code you paste, run, and save is stored only locally on your device.
- The names and icons you assign to saved apps are stored only locally on your device.
- We have no servers and never upload any of the above to the cloud or any third party.
- We do not collect your name, email, location, device identifiers, or any personally identifiable information.
- We do not use any third-party analytics, advertising, or tracking SDKs.

### How Data Is Stored

- All your data (HTML content, app names, custom icons) is stored within the app's sandbox on your device.
- This data never leaves your device unless you **actively** use the "Share" feature to export a saved app as a file to someone else.
- When you uninstall the app, all local data is deleted along with it.

### About Running HTML Content

- The app runs the HTML code you provide via the system WebView component.
- The app itself does not initiate network connections. However, if the HTML you run contains links or resources pointing to the network (e.g., external images or scripts), running that code may generate corresponding network requests — this depends on the HTML content you run and is unrelated to the app itself.
- Please only run HTML code from sources you trust.

### Share & Import

- The "Share" feature is initiated by you, packaging a chosen app into a `.pocket` file and sending it via a method you select (e.g., AirDrop, messaging apps). We do not participate in, handle, or store any data during this transfer.
- The "Import" feature, when you open a `.pocket` file sent by others, asks for your confirmation before importing, and only saves the content to your device locally after you agree.

### Children's Privacy

This app is not directed at children under 13 and does not knowingly collect any personal information from children.
